## 📊 Core Algorithm: S-Algorithm
The core logic calculates the "Psychological Frictionless Expenditure Limit" ($S$) for end-users to prevent checkout hesitation.

$$S = \frac{I - (F+H)}{D}$$

- **I**: Monthly Disposable Income
- **F+H**: Fixed Costs + Hidden Friction Costs
- **D**: Remaining Days in Month
- **S**: Daily Safe Expenditure Limit
